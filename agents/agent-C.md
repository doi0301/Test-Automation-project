## TC 생성 핵심 포인트

### FZ0341 (권리유지 검토) — 4가지 분기 전부 TC 필수

플로우차트 기반:

| 선택 | 지시여부 | 전환 상태 | TC 포인트 |
|------|---------|---------|---------|
| 납부지시 | Y | 검토완료 → 미납부 | 전송 확인 |
| 타사납부 | N | 검토완료 → 납부포기 | 타사 처리 확인 |
| 타사이관 | - | 접수취소 전환 | 타사이관 처리 확인 |
| 납부포기 | N | 납부포기 전환 | 권리포기 확인 |

### FZ029 (리마인더회신) — 권한 TC 집중

리마인더 처리는 출원일반만 가능:
- 협력일반/연차일반 처리 성공 TC 금지 (forbidden-patterns 참조)

### 리마인더 재발송 케이스

- WHEN 회신일 경과 → 자동 재발송 TC (Mock 필요)
- WHEN 오청구 → 청구 재발송 + 미납부 상태 유지 TC
