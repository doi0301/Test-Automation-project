- **처리권한 X인 경우**: 모든 버튼 미출력 (열람도 불가)
- **협력담당자 처리 조건**: 협력담당자 본인이거나 그룹원이 협력담당자인 경우

## TC 적용 시 주의사항

1. **협력담당자 범위**: "협력담당자인 경우 또는 그룹원이 협력담당자인 경우" — 두 조건 각각 TC 필요
2. **유지료담당자 범위**: "유지료담당자인 경우 또는 그룹원이 유지료담당자인 경우" — 두 조건 각각 TC 필요
3. **TASK불러오기**: 출원담당자는 전체 TASK / 협력담당자는 협력담당자인 TASK만 → 범위 차이 TC 필수
4. **리마인더 처리**: 출원일반만 가능 — 협력담당자/유지료담당자 처리 성공 TC 금지
